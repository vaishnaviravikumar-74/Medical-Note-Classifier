# Medical Note Classifier

A machine learning application that classifies medical notes into medical specialties (e.g., Cardiology, Neurology, Orthopedics, etc.).

This project preprocesses clinical notes, fine-tunes a `bert-base-uncased` model for text classification, and provides a terminal-based interface to perform predictions on new notes.

## Setup & Installation

1. Ensure Python is installed (preferably in a virtual environment).
2. Install dependencies by running:
```bash
pip install -r requirements.txt
```

## 1. Data Processing
Place your raw clinical text dataset inside `data/mtsamples.csv`. This CSV must contain a text column and a label column (e.g. `transcription` and `medical_specialty`).

Run the preprocessor to clean the dataset:
```bash
python src/preprocess.py
```
This generates `cleaned_data.csv` and `label_mapping.csv` under the `data/` directory.

## 2. Model Training & Evaluation
After preprocessing, execute the training pipeline. This script trains `.bert-base-uncased` on `cleaned_data.csv` and spits out evaluation metrics.

```bash
python src/train_model.py
```
* Note: Depending on hardware, fine-tuning BERT can be resource-intensive.
* The fully trained model artifacts are dumped inside `/models`.
* The evaluation metrics (Accuracy, Precision, Recall, F1) are saved to `results/evaluation_report.txt`.

## 3. Terminal Prediction
Once a model has been successfully trained and placed in the `/models` directory, you can interact with it via the CLI:

```bash
python src/predict.py
```
